# Released or Not

### What is it?
*Released or Not* is a simple script that helps you manage your Animes.  

## Goals
- [x] Backup and shareable list
- [x] Integrated searching tool
- [x] Option to run 24/7
- [x] Show accumulated released episodes
- [x] Add synopses
